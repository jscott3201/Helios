# M09 KV Compression Research Evidence

This file records the local M09 work completed on 2026-06-30.

## Status

M09 is implemented as fixture-based KV compression research in `gemma4d-kv`. BF16 remains the default fallback, q8/q4 affine prefix-cache modes are namespace-scoped and manifest-visible, Planar/Iso candidates are feature-gated and not accepted by default, and M09 evidence records quality plus memory/speed comparisons for q8/q4 across 16K, 32K, and 64K fixture workloads.

## Implemented

- Added q8/q4 cache-mode helpers and byte estimates.
- Added `CompressionManifestMetadata` to persisted KV manifests and the JSON schema.
- Added deterministic q8/q4 affine quality evaluation over fixture logits.
- Added namespace/block-ID isolation tests for BF16, q8, and q4 modes.
- Added feature-gated `planar-iso-experiments` interface with Planar4, Planar3, Iso4, and Iso3 candidates rejected by default.
- Added `m09_compression_eval` evidence runner.
- Added TUI cache-page compression summary for q8/q4 cosine, memory reduction, BF16 default, namespace isolation, and Planar/Iso status.
